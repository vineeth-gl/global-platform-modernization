# Changelog — 15+ years of evolution (abbreviated)

## 2010 — ShopForge launch
Initial monolith (later Java 8 port). Single SQL DB.

## 2012 — First internationalization
EU expansion. jQuery portal becomes primary ops console.

## 2014 — AcmeBill acquisition
Billing platform acquired. Order ID floor raised to 900000.

## 2016 — CloudPeak vendor engagement
Partial AWS migration. Secrets copy-pasted into repos.

## 2018 — Mongo VIP store on Azure
Split-brain customers begin. Sync cron on VM.

## 2019 — Inventory extracted to Node microservice
Circular hooks to monolith retained "temporarily".

## 2020 — React portal pilot
Legacy portal still required for catalog ops.

## 2021 — Kafka + Rabbit dual publish
Compatibility layer; removal deferred yearly.

## 2023 — Canary + blue-green release patterns
Weekly releases + hotfix pipeline. SLA 99.95%.

## 2025–2026 — Zero Trust WIP
Feature flag exists; VM fleet still flat network.
